API : Timestamp


API ENDPOINT :

GET /api/timestamp/:datestring

GET /api/timestamp (IT WILL GIVE CURRENT TIMESTAMP)


datestring FORMAT :

yyyy-m-d (2014-10-5)
yyyy-m (2014-10)
d-m-yyyy (5-10-2014)
Unix time (1412121600000)

JSON Object :

{ "unix" : <int:unix-time>, "utc" : <string:utc-date> }

Try :

/api/timestamp/2014-5
/api/timestamp/1398902400000
/api/timestamp
